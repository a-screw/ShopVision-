# ShopVision
Deep learning project using CNNs + ResNet to classify product images. Helps e-commerce platforms automate product tagging, improve search relevance, and reduce manual effort. Built with TensorFlow/Keras.



🧠 CNN-Based Product Image Classification
This project implements a Convolutional Neural Network (CNN) model to automatically classify product images into categories. It leverages transfer learning with ResNet to improve accuracy and reduce training time.
🚀 Overview
Manual product categorization in e-commerce is time-consuming and error-prone. This project builds an AI-powered system that can:
Automatically classify product images
Improve product search and filtering
Reduce manual tagging efforts
🏗️ Model Architecture
📌 Pretrained ResNet model (transfer learning)
📌 Custom fully connected classification head
📌 Fine-tuning for improved performance
